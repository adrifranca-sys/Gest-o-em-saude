Acesso e login no sistema 6 situações
acesso Usuário não consegue fazer login — "usuário ou senha inválidos" ▼
⚠️ Verificar antes de qualquer ação se o usuário está acessando o ambiente correto (produção x homologação).
  1. Confirmar o CPF cadastrado — sem pontos ou traços.
  2. Orientar a tentar a opção "Esqueci minha senha" na tela de login.
  3. Checar se a conta está bloqueada por tentativas (acima de 5 tentativas o sistema bloqueia por 30 min).
  4. Se persistir, abrir chamado para o setor de TI com CPF e unidade do usuário.
💡 O desbloqueio manual só pode ser feito pelo administrador do sistema. Tempo estimado: até 2h úteis.
acesso Usuário diz que a senha expirou e não sabe como redefinir ▼
ℹ️ Senhas expiram a cada 90 dias por política de segurança.
  1. Orientar acesso ao link /resetar-senha na tela inicial do sistema.
  2. O e-mail de redefinição vai para o endereço cadastrado no perfil. Confirmar se o usuário tem acesso a esse e-mail.
  3. Caso não tenha acesso ao e-mail, escalar para o gestor da unidade para atualização do cadastro.
⚠️ Nunca forneça ou redefina senhas diretamente pelo chat. Sempre redirecionar para o fluxo padrão de redefinição.
acesso Usuário logou mas não vê o menu / funcionalidades esperadas ▼
  1. Verificar o perfil de acesso atribuído ao usuário (regulador, solicitante, visualizador, etc.).
  2. Confirmar se o usuário está vinculado à unidade correta no cadastro.
  3. Checar se há manutenção programada que possa ter impactado permissões.
  4. Se o perfil estiver correto e o problema persistir, registrar ocorrência com print de tela.
💡 A atribuição ou alteração de perfil deve ser feita pelo gestor da unidade, não pelo suporte.
acesso Sistema apresenta mensagem de erro ao abrir — tela branca ou travamento ▼
⚠️ Verificar antes se há algum aviso de indisponibilidade no canal oficial do sistema.
  1. Solicitar que o usuário limpe o cache do navegador (Ctrl+Shift+Del) e tente novamente.
  2. Testar em outro navegador (recomendado: Chrome ou Edge atualizados).
  3. Checar se há extensões bloqueando scripts (AdBlock, etc.).
  4. Se o problema for generalizado, acionar equipe de infraestrutura imediatamente.
acesso Usuário recém-cadastrado não consegue acessar pela primeira vez ▼
  1. Confirmar se o cadastro foi concluído e aprovado pelo gestor (usuários pendentes não conseguem logar).
  2. Verificar se o e-mail de ativação foi enviado — pode estar na pasta de spam.
  3. O link de ativação expira em 24 horas. Se expirado, solicitar reenvio ao gestor.
  4. Confirmar que o usuário está definindo senha dentro dos requisitos: mínimo 8 caracteres, 1 maiúscula, 1 número e 1 símbolo.
acesso Usuário ativo na unidade foi desligado — como revogar acesso? ▼
🚨 Caso urgente: usuário com acesso indevido deve ser suspenso imediatamente. Escalar com prioridade alta.
  1. Gestor da unidade deve acessar Gerenciar Usuários → Inativar.
  2. Se o gestor não tiver acesso para isso, abrir chamado de alta prioridade para administrador do sistema.
  3. Registrar no chamado: nome, CPF, unidade e data do desligamento.
ℹ️ Usuários inativos não são excluídos — o histórico de ações fica preservado para auditoria.

Indeferimento de solicitações 6 situações
indeferimento Paciente com mais de um pedido para o mesmo procedimento — indeferimento por duplicidade ▼
ℹ️ Aplicar quando identificado que um paciente possui mais de um pedido para o mesmo procedimento, evitando duplicidade na fila de regulação.

Passo 1 — Localização da solicitação

  1. No menu lateral esquerdo, acesse a aba Solicitações e clique em Buscar.
  2. Utilize o campo CPF do Paciente para filtrar todos os pedidos vinculados àquele usuário.
  3. Clique no botão azul Buscar no canto direito.

Passo 2 — Identificação da duplicidade

  1. Na listagem, observe as colunas Id e Data.
  2. Identifique qual solicitação será mantida e qual será indeferida — geralmente mantém-se a mais antiga ou a enviada via E-sus PEC, conforme a regra de negócio.
  3. Clique no botão azul Regulado (ou no ID correspondente) da solicitação que deseja cancelar.

Passo 3 — Execução do indeferimento

  1. Uma janela suspensa será aberta: "Indeferir procedimentos para [Nome do Paciente]".
  2. Marque as caixas de seleção ao lado de cada procedimento a ser indeferido (ex: Ecocardiografia, Eletrocardiograma).
  3. No campo Observações, digite o motivo padrão: Indeferimento solicitado via chat suporte por motivo de duplicidade.
  4. Clique no botão vermelho Indeferir para concluir.
🚨 Esta ação é irreversível. Confirme os dados do paciente e os procedimentos antes de clicar em Indeferir.
indeferimento Solicitação indeferida por "documentação incompleta" — o que fazer? ▼
ℹ️ O indeferimento por documentação não cancela definitivamente a solicitação — permite recurso.
  1. Verificar no histórico da solicitação quais documentos foram apontados como faltantes.
  2. Orientar o solicitante a reunir a documentação indicada.
  3. Acessar a solicitação e usar a opção "Entrar com Recurso" para reenvio com os documentos.
  4. Prazo para recurso: até 30 dias a partir da data do indeferimento.
✅ Informar ao solicitante que o recurso será reanalisado em até 10 dias úteis após o envio.
indeferimento Solicitação indeferida por "fora do perfil assistencial" da unidade ▼
  1. Explicar que a solicitação requer um serviço que não está habilitado na unidade solicitante.
  2. Verificar no sistema qual unidade tem habilitação para o procedimento necessário.
  3. Orientar o reencaminhamento para a unidade adequada ou para o regulador responsável redirecionamento.
⚠️ Esse indeferimento não admite recurso direto — é necessário nova solicitação pela unidade habilitada.
indeferimento Solicitação indeferida por "CID não compatível com o procedimento" ▼
⚠️ Este é um dos indeferimentos mais frequentes e geralmente tem solução simples.
  1. Verificar na solicitação qual CID foi informado e qual procedimento foi solicitado.
  2. Consultar a tabela de compatibilidade CID x procedimento disponível no módulo de referências do sistema.
  3. Orientar o médico solicitante a corrigir o CID ou justificar a indicação clínica na solicitação.
  4. Após correção, utilizar "Entrar com Recurso" com o laudo/justificativa atualizado.
indeferimento Paciente quer saber o motivo do indeferimento — como informar? ▼
  1. Acessar a solicitação e localizar a aba "Histórico / Pareceres".
  2. O motivo detalhado fica registrado pelo regulador no momento do indeferimento.
  3. Repassar ao paciente o motivo oficial, de forma clara e acessível.
  4. Informar sobre o direito ao recurso e o prazo disponível.
💡 O paciente também pode consultar o status pelo portal do cidadão usando CPF + número da solicitação.
indeferimento Recurso foi negado novamente — quais são as próximas instâncias? ▼
🚨 Quando o recurso é negado em segunda instância, o caso passa a ser administrativo — fora do escopo do suporte técnico.
  1. Informar que o suporte técnico não tem alçada para reverter decisões clínicas ou administrativas.
  2. Orientar o solicitante a acionar a Ouvidoria da Secretaria de Saúde responsável.
  3. Caso haja indicação de urgência ou risco de vida, orientar acesso via regulação de urgência (fluxo diferenciado).
  4. Registrar o atendimento no sistema de chamados com todos os dados para rastreabilidade.
